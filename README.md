# xdicabot
Cara install bot WhatsApp XDica BOT

pertama-tama

1. pkg update && pkg upgrade
2. pkg install git -y
3. pkg install nodejs-lts
4. pkg install ffmpeg -y
5. git clone https://github.com/rezonexd/xdicabot
6. cd xdicabot
7. yarn
8. npm start

(angka dan titik tidak di bawa)
